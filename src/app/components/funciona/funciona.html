¿Cómo funciona?

Realizamos el procedimiento con estos sencillos pasos

Nota previa

Es importante que el dispositivo Hikvision y el computador estén conectados a través de cable o WiFi a la misma red.

Paso 1

Debes tener descargado e instalado el software Anydesk en tu computador.

Paso 2

Indícanos tu ID de Anydesk y acepta la conexión remota que enviaremos.

Paso 3

Iniciamos el proceso de restablecimiento con archivo seguro, este paso tiene una duración máxima de 40 minutos.

Nota: Realizar un restablecimiento de clave no afectará el funcionamiento del dispositivo ni se perderá información alguna.

Paso 4

Confirmamos el cambio de contraseña ingresando a la versión web del dispositivo o agregándolo al software Batch Configuration.

Nota final

Tras restablecer el dispositivo, se asignará una contraseña provisional conocida únicamente por nuestro equipo.

La clave definitiva proporcionada por el cliente será configurada una vez confirmado el pago del servicio.
